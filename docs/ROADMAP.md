# Roadmap

## v1

- manual incident markers
- PresentMon-backed frame telemetry when configured
- system/process/network telemetry
- OBS integration via websocket
- WPF tray app with hotkeys
- ring buffer and incident materialization
- ZIP export with redaction defaults
- fake scenarios and acceptance tests

## v2

- richer FiveM artifact import coverage
- stronger endpoint mapping and session affinity for FiveM servers
- better RTT/jitter sampling strategies
- rule engine v2 with cross-incident learning
- improved resource/script attribution from profiler and resmon artifacts
- richer timeline visualizations

## v3

- deeper ETW/WPR analysis
- better scheduler/DPC/ISR attribution
- stronger disk/streaming classification
- more advanced reports and shareable evidence packages
- repeated-incident clustering and cache/resource-corruption heuristics
